인트라넷 웹 체계 유지보수 업무
개발 환경
  • Eclipse(IDE)
  • Apache Tomcat(WAS)
  • Informix(Database)
사용 기술
  • HTML, CSS를 사용하여 웹 페이지 구조 설계 및 레이아웃 조정
  • JQuery를 활용하여 DOM 조작, 이벤트 처리, AJAX
  • JSP 동적 웹 페이지 작성
  • 데이터베이스 연결 및 SQL 쿼리 작성
역할
  • 국방망을 통해 운영되는 웹 체계의 유지보수를 담당
  • 소요 제기 담당자와의 의사 소통을 통해 프로젝트의 요구 사항을 파악 후 기능 개선
  • 타 개발자가 작성한 코드를 분석 후 수정 및 보일러플레이트 코드 리팩토링
전자정부프레임워크 교육 이수
기존의 프로젝트
  • 개발에 필요한 JAR 라이브러리를 수동 설치
  • Scriptlet 방식으로 하나의 JSP 페이지 안에서 모든 로직이 수행되도록 작성
  • JDBC 드라이버로 Connection 생성, PreparedStatement 생성 후 파라미터 바인딩, ResultSet에 쿼리 결과 받아온 후 객체에 수동으로 맵핑
전자정부프레임워크
  • Maven으로 필요한 라이브러리와 그 라이브러리에 종속되는 다른 라이브러리를 자동으로 받아와서 빌드
  • Spring MVC 패턴을 도입해서 요청을 Controller에서 받아 Service 로직을 수행하고 그 결과를 Model로 받아와서 View에 전달한 후 JSTL로 출력
  • MyBatis로 SQL을 XML로 관리하며 파라미터와 결과값이 객체에 자동 맵핑
